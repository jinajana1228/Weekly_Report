# V1 시장 분석 로직 설계 문서

> **문서 목적**: 매주 발행하는 1개월형 주간 추천 리포트의 시장 분석 섹션을 어떤 구조와 로직으로 작성할지 정의한다.
> 이 문서는 구현 전 설계 검토 단계의 산출물이다. 실제 데이터 소스 확정, 코드 구현, 가중치 수치 확정은 이 문서에서 하지 않는다.

---

## 1. 시장 분석의 목적 (이 서비스 맥락에서)

이 서비스의 시장 분석은 두 가지 역할을 수행한다.

1. **맥락 제공**: 독자(투자자)가 이번 주 추천 종목을 어떤 시장 환경 위에서 판단해야 하는지 배경을 제공한다.
2. **추천 로직의 기반**: 어떤 섹터가 이번 2~4주에 유리하고 어떤 섹터가 주의가 필요한지(`favored_sectors` / `cautious_sectors`) 결정하는 입력값이 된다.

**중요한 제약**: 이 서비스는 1개월(2~4주) 관점이다. 시장 분석 문장은 3~6개월 장기 방향이 아니라, **지금 이 주에 유효한 2~4주짜리 시장 국면**을 설명해야 한다.

---

## 2. 글로벌 시장 분석 프레임

### 레이어 구분

| 레이어 | 분석 대상 | 1개월형에서의 역할 |
|--------|----------|-----------------|
| L1. 글로벌 매크로 방향 | Fed 금리 방향, 달러 강세/약세 기조, VIX(공포지수) 수준, 글로벌 위험선호(Risk-on/off) | 이번 주 시장 전체 분위기의 배경 설명. 급격한 기조 전환 여부 확인. |
| L2. 주요 지수 주간 등락 | S&P500, NASDAQ, Nikkei225, CSI300의 주간 등락률 및 방향성 | 외부 변수가 국내 시장에 미치는 1차 영향 파악. 글로벌 동조화 여부. |
| L3. 글로벌 이벤트/일정 | FOMC 발표, 미국 CPI/PPI, 글로벌 실적 시즌, 지정학적 이벤트 | 2~4주 내 시장 방향을 바꿀 수 있는 잠재 변수 파악. |
| L4. 글로벌 섹터 스타일 | 미국 기술주 vs 경기방어주 흐름, 성장주 vs 가치주 로테이션 | 국내 동일 섹터에 외부 스타일 흐름이 전이될 가능성 판단. |

### 1개월형에 적합한 이유

- L1(매크로 기조)은 1개월 내 급격히 변하지 않는 배경 변수이므로 분석 비중이 낮다. 기조 자체보다 **기조 변화 신호**가 있는지가 핵심이다.
- L3(이벤트/일정)는 2~4주 내 실현되는 이벤트만 포착한다. 3개월 이후 이벤트는 이번 분석 범위 밖이다.
- L4(스타일 흐름)는 국내 동일 섹터 추천 판단에 직접 영향을 주므로 주간 단위로 확인한다.

## 3. 국내 시장 분석 프레임

### 레이어 구분

| 레이어 | 분석 대상 | 1개월형에서의 역할 |
|--------|----------|-----------------|
| L1. 국내 거시 환경 | 원/달러 환율 방향, 국내 기준금리 기조, 무역수지, 국내 경기지표 | 국내 시장 전반의 배경. 환율은 수출주/해외ETF에 직접 영향. |
| L2. 국내 지수 수급 | 코스피/코스닥 외국인·기관·개인 순매수 흐름, 프로그램 매수 | 이번 주 실제 수급 방향. 섹터 선택에 가장 직접적인 신호. |
| L3. 국내 스타일 흐름 | 대형주 vs 소형주, 성장주 vs 배당주, 모멘텀 vs 저평가 | 추천 종목의 크기/스타일이 현재 시장 기류와 맞는지 판단. |
| L4. 업종 상대강도 | 각 섹터(13개 코드 기준)의 코스피 대비 주간 초과수익률 | `favored_sectors` / `cautious_sectors` 도출의 핵심 입력. |
| L5. 국내 이벤트/일정 | 국내 기업 실적 발표 일정, 국내 경제지표 발표, 정책 발표 | 2~4주 내 특정 섹터 또는 종목에 영향을 줄 단기 이벤트. |

### 1개월형에 적합한 이유

- L2(수급)와 L4(업종 상대강도)는 주간 단위로 변화가 빠르므로 1개월형 서비스에서 가장 비중이 높다.
- L1(거시 환경)은 배경으로만 활용하고, 실제 추천 판단은 L2~L5에 더 의존한다.
- L5(국내 이벤트)는 2~4주 내 가시적 촉매의 핵심 입력이다. 이 레이어가 약하면 추천 로직의 촉매 근거가 부실해진다.

## 4. 시장 분석에 필요한 데이터 레이어 구분

### 공식 수치 데이터 (정량)

| 데이터 항목 | 용도 | 확정 여부 |
|------------|------|----------|
| 주요 지수 주간 등락률 (S&P500, KOSPI 등) | 지수 방향 확인 | ✅ 활용 방향 확정 |
| 외국인/기관/개인 순매수 금액 | 수급 흐름 판단 | ✅ 활용 방향 확정 |
| 원/달러 환율 주간 변동 | 국내 환경 배경 | ✅ 활용 방향 확정 |
| 섹터별 주간 등락률 (KRX 또는 자체 집계) | 업종 상대강도 계산 | ⚠️ 데이터 소스 미확정 |
| VIX 수준 | 글로벌 위험선호 판단 | ⚠️ 데이터 소스 미확정 |
| 미국 CPI/FOMC 결과 등 | 글로벌 이벤트 확인 | ⚠️ 수집 방식 미확정 |

### 뉴스 신호 데이터 (정성)

| 신호 유형 | 용도 | 원칙 |
|----------|------|------|
| 업종 방향 뉴스 | 특정 섹터 긍정/부정 뉴스 흐름 감지 | 기사 전문이 아닌 **방향성 신호**로만 활용 |
| 이슈 기업 뉴스 | 실적 발표, M&A, 규제, 제품 출시 등 | 종목 추천의 촉매 판단에 활용. 기사 재배포 금지. |
| 정책/규제 뉴스 | 국내 산업 정책, 세제 변화, 규제 방향 | 섹터 유불리 판단 보조 신호. 공식 지표로 검증 필요. |
| 글로벌 매크로 뉴스 | Fed 발언, 지정학적 이벤트 등 | 단순 요약이 아닌 "이번 주 시장에 어떤 영향인지" 구조화 필요. |

## 5. 공식 수치 데이터 vs 뉴스 신호 데이터 역할 분리 원칙

| 구분 | 역할 | 사용 위치 |
|------|------|----------|
| **공식 수치 데이터** | 시장 상황의 객관적 사실 확인. 수치 자체가 방향 판단의 1차 근거가 된다. | 시장 분석 문장의 기반. `favored_sectors` 도출의 1차 근거. |
| **뉴스 신호 데이터** | 수치 데이터가 포착하지 못하는 방향성 변화 단서를 보완한다. 확인되지 않은 신호는 주장이 아닌 "신호 감지" 수준으로만 기술. | 시장 분석 문장의 맥락 보완. 종목 촉매 판단의 보조 입력. |
| **우선순위 원칙** | 수치와 뉴스 신호가 충돌하면 수치 데이터를 우선한다. | 전체 분석 과정 |

### 뉴스 신호 활용 시 금지 사항

- 특정 기사의 내용을 직접 인용하여 재배포하는 것은 금지한다.
- 단일 기사 1개만을 근거로 섹터 방향 판단을 내리지 않는다.
- 출처가 불명확한 루머성 뉴스를 분석 근거로 사용하지 않는다.
- 뉴스 신호는 항상 "공식 수치 데이터로 검증 가능한가"를 먼저 확인한다.

## 6. favored_sectors / cautious_sectors 도출 판단 구조

### 도출 원칙

`favored_sectors`(유리한 섹터)와 `cautious_sectors`(주의 섹터)는 단일 신호 하나가 아니라, 아래 **3개 축의 교집합**으로 판단한다.

```
판단 축 1: 수급 신호
  → 외국인/기관 순매수가 집중되는 업종인가?
  → 직전 주 대비 수급 방향이 전환되었는가?

판단 축 2: 업종 상대강도
  → 코스피 대비 섹터 주간 초과수익률이 양수인가?
  → 최근 2~3주 누적 상대강도가 지속적으로 개선되는 추세인가?

판단 축 3: 이벤트/촉매 일정
  → 이번 2~4주 내 해당 섹터에 긍정적 이벤트(실적, 정책, 제품 출시 등)가 예정되어 있는가?
  → 반대로 부정적 이벤트(규제, 실적 악화 신호 등)가 예정되어 있는가?
```

### 도출 등급 (확정 필요)

> ⚠️ **미확정**: 아래는 설계 제안이며, 실제 운영 기준은 추가 검토 필요.

| 등급 | 설명 | 예시 표현 |
|------|------|----------|
| favored | 3개 축 중 2개 이상이 긍정적 | "수급 유입 + 이벤트 일정 겹침" |
| neutral | 신호가 혼재하거나 특이사항 없음 | 별도 기재 없음 |
| cautious | 3개 축 중 2개 이상이 부정적 | "외국인 순매도 + 규제 이슈" |

### 주의사항

- 섹터가 `cautious`로 분류되어도 해당 섹터 내 개별 종목 추천이 완전히 금지되는 것은 아니다. 다만 추천 사유가 더 강력해야 한다.
- `favored_sectors`로 분류된 섹터에서 반드시 추천 종목을 뽑아야 하는 것도 아니다. 시장 분위기와 추천은 상관관계이지 결정 관계가 아니다.

## 7. 1개월형 서비스에 맞는 시장 요약 작성 원칙

### 작성 구조 (3요소)

1개월(2~4주) 관점의 시장 분석 문장은 아래 3가지 요소를 반드시 포함해야 한다.

| 요소 | 설명 | 예시 방향 |
|------|------|----------|
| **현재 국면 정의** | 지금 시장이 어떤 상태인지 1~2문장으로 규정한다. | "외국인 순매수 전환 + 기술주 반등 초입 구간" |
| **핵심 변수 1~2개** | 앞으로 2~4주 안에 시장 방향을 가를 수 있는 가장 중요한 변수를 특정한다. | "FOMC 결과 해석 방향", "반도체 수출 지표" |
| **주의 요인 1개** | 예상이 빗나갈 경우의 시나리오 또는 하방 리스크를 기술한다. | "원/달러 환율 재상승 시 외국인 이탈 가능성" |

### 금지 표현 원칙

| 금지 유형 | 예시 | 대체 표현 방향 |
|----------|------|--------------|
| 단정적 방향 예측 | "이번 주는 반드시 반등한다" | "반등 시도 가능성이 높은 구간" |
| 과도한 낙관 | "지금이 매수 기회다" | "관심 구간 진입 여부를 점검할 시점" |
| 근거 없는 공포 조장 | "큰 조정이 올 수 있다" | "외부 변수에 따른 변동성 확대 가능성 존재" |
| 3개월 이상 장기 전망 | "하반기에는 강세장이 예상된다" | 1개월형 서비스 범위 초과. 기술 금지. |

### 글로벌 vs 국내 시장 문장 비중 원칙

- 글로벌 시장 문장: 1~2문장. 배경 역할에 집중. 국내 시장과의 연결 고리를 명시.
- 국내 시장 문장: 3~4문장. 수급, 섹터 흐름, 이번 주 핵심 이벤트 중심.
- 비중 원칙: 국내 시장 분석이 글로벌 분석보다 항상 더 길고 구체적이어야 한다.

## 8. 시장 분석 항목 확정 현황

### 확정된 원칙

| 항목 | 확정 내용 |
|------|----------|
| 분석 관점 기간 | 2~4주 (1개월형) |
| 분석 레이어 수 | 글로벌 4개 레이어, 국내 5개 레이어 |
| 수치 vs 뉴스 우선순위 | 수치 데이터 우선, 뉴스 신호 보완 역할 |
| favored/cautious 도출 축 | 수급 신호, 업종 상대강도, 이벤트/일정 3축 교집합 |
| 뉴스 활용 원칙 | 기사 재배포 금지, 구조화된 방향 신호로만 활용 |
| 시장 요약 3요소 | 현재 국면 정의 + 핵심 변수 + 주의 요인 |
| 금지 표현 유형 | 단정적 예측, 과도한 낙관, 3개월 이상 장기 전망 |

### 정책 확정이 더 필요한 항목

| 항목 | 미확정 이유 | 확정 필요 시점 |
|------|-----------|--------------|
| 섹터별 상대강도 데이터 소스 | KRX 공식 API, 데이터 벤더, 직접 집계 방식 미결 | 데이터 소스 설계 단계 |
| VIX 수치 수집 경로 | 수집 자동화 방식 미결 | 데이터 수집 파이프라인 설계 시 |
| favored/cautious 도출 최소 조건 수치화 | "2개 축 이상 긍정"의 정량 기준 미결 | 실제 운영 1~2회 후 보정 |
| 시장 분석 문장 작성 주체 | 자동 생성 vs 수동 작성 vs AI 보조 후 수동 검토 | 운영 방식 결정 시 |
| 국내 수급 데이터 수집 주기 | 주간 집계 기준 시점(금요일 장마감 기준 등) 미결 | 데이터 수집 파이프라인 설계 시 |

---

> 이 문서는 V1 시장 분석 로직의 설계 기준이다.
> 실제 데이터 수집 방식, 자동화 구현, 가중치 수치는 이 문서에서 확정하지 않는다.
